# Sales Prediction Project

This project analyzes sales data, performs visualizations, applies regression, clustering, outlier detection, and time series decomposition using Python.

## Features
- Data transformation & scaling
- Statistical analysis (Chi-square, Z-test)
- Visualization (Lineplot, Barplot, Pie, Heatmap, Polar, etc.)
- Regression analysis
- Outlier detection (IQR & Z-score)
- KMeans clustering
- Time series decomposition

## Libraries Used
- pandas, numpy, matplotlib, seaborn
- scikit-learn, scipy, statsmodels
